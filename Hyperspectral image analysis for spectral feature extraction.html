Choosing a Region of Interest (ROI)¶

The next step was to choosse a region of interest on the sample. This can be denoted by selecting an origin on the 2-D array by assigning co-ordinates to x and y.

The w parameter sets the horizontal limit. w extends out to the right of the origin. The h parameter sets the vertical limit of the region. h extends downwards from the origin.

line and intensity are visual parameters and only serve purpose to make it easier to view the selected region

In [3]:
def extract_roi(arr, x, y, w, h, intensity, line):
    roi = arr[y:y+h, x:x+w, :]

    bounding_box = arr
    bounding_box[y-line:y, x-line:x+w+line, :] = intensity 
    bounding_box[y:y+h, x-line:x, :] = intensity 
    bounding_box[y+h:y+h+line, x-line:x+w+line, :] = intensity
    bounding_box[y:y+h, x+w:x+w+line, :] = intensity

    return (roi, bounding_box)

Plotting the mean absorabance of a Region of Interest¶

plotROI() is a function used to plot the mean absorabance spectrum within the region of interest.

It takes the mean transmittance value accross every pixel in the region of interest, and this transmittance value is converted to a measure of absorbance.

def plotROI(roi,eV,title):
    #fig2, ax2 = plt.subplots(figsize = (10,7))
    intensity = []
    intensity_corrected =[]

        intensity.append(np.mean(roi[:, :,b]))

def pixel_spectrum(x,y,arr):
    pixel_x = x
    pixel_y = y
    pixel = arr[pixel_y, pixel_x,:]
    pixel_abs = - np.log10(pixel)
    pixel_abs = pixel_abs
    return pixel_abs

Calculating the Average Free Exciton Width¶

The average free exciton width of the sample can be estimated from the absorption ratio between A(0-0) and A(0-1).

The ratio parameter is the ratio of the absorbance at 1.5 eV and the absorbance at 2.3 eV.

In [6]:
def exciton_width(ratio):
    w = (1 - math.sqrt(ratio))/((0.405555*math.sqrt(ratio))+1.3333333)
    return w

ew = exciton_width(ratio)

Identifying the peaks in the absorbance spectrum¶

Using peaks() the peak value of the absorbance spectra at a pixel is extracted using the SciPy find_peaks function.

The absorbance and the corresponding wavelength are stored to y and x, respectively, for visualisation.

In [10]:
def peaks(pixel_abs,eV):
    x = np.stack((eV[14:],pixel_abs[14:]),1)
    x=x.T
    peaks , _ = scipy.signal.find_peaks(x[1],distance = 300)
    peakabs = x[1][peaks]
    peakindex = np.where(x[1] == peakabs)
    peakenergy =x[0][peakindex[0]]
    return peakabs,peakenergy

y,x = peaks(individual,eV)

plt.plot(x,y,'x',color = 'black')

Bringing together all functions to create characteristic heatmaps¶

peakabs() collates all the above the functions into one function to take a region of interest (ROI) and get the peak absorbance, corresponding wavelength and the free exciton width for each pixel within the region.

These values are stored as 2D arrays which are then plotted as heatmaps using the library, seaborn.

In [ ]:
def peakabs(rois,eV,el):
    abslist = []
    evlist =[]
    energyratio=[]
    abs = np.zeros((roi.shape[0],roi.shape[1]))
    energy = np.zeros((roi.shape[0],roi.shape[1]))
    ratio = np.zeros((roi.shape[0],roi.shape[1]))
    
    for x in range(roi.shape[0]):
        for y in range(roi.shape[1]):
            spectra = pixel_spectrum(x,y,roi)
            ratio[x][y] = spectra[el]/spectra[33]
            peakabs,peakev = peaks(spectra,eV)    
            peakabs[peakabs>10000000] =0
            peakabs[peakabs>1] = np.mean(peakabs)
            abs[x][y]=peakabs.max()
            energy[x][y]=peakev.max()    
        abslist.append(abs)
        evlist.append(energy)
        energyratio.append(ratio)
    return abslist, evlist, energyratio

absmap, evmap, ratiomap = peakabs(roi,eV,210)
